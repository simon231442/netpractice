# NetPractice

## Description

Brief overview: this project provides a set of web-based exercises to learn basic
computer networking concepts and to practice configuring small simulated
networks (IP addressing, gateways, routers, switches).

## Instructions

Open `index.html` in a web browser such as Chrome, Brave, or Vivaldi. To access a specific exercise directly, open the file corresponding to the level you want.

## Submission

- Place the 10 exported configuration files (one per level) at the root of the repository before submission.
- During the defense, you will be asked to complete 3 random levels within 15 minutes.

## Resources

- A GitBook to understand the theory: https://github.com/lpaube/NetPractice
- An IP calculator to check ranges: https://www.calculator.net/ip-subnet-calculator.html
- Peer learning with other students was very helpful.
